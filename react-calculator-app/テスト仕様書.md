# 電卓アプリ テスト仕様書

## 1. 機能要件テスト

### 1.1 表示欄
- [X] 初期表示は「0」となっていること
- [ ] 入力した数字・演算子が表示欄に反映されること
- [ ] 計算結果が表示欄に反映されること

### 1.2 数字ボタン(0~9)
- [ ] 各数字ボタンを押すと、表示欄にその数字が追加されること

### 1.3 演算子ボタン(+, -, *, /)
- [ ] 演算子ボタンを押すと、表示欄にその演算子が追加されること

### 1.4 クリアボタン(C)
- [ ] クリアボタンを押すと、表示欄が「0」になること

### 1.5 イコールボタン(=)
- [ ] 正しい式（例：1+2）でイコールを押すと、計算結果が表示されること
- [ ] 「整数 演算子 整数」以外の形式（例：123+、1+*2）でイコールを押すと、「エラー」と表示されること
- [ ] 0で割る式（例：1/0）でイコールを押すと、「エラー」と表示されること

## 2. 入力パターンテスト

| 入力内容 | イコール押下後の期待結果 |
|----------|------------------------|
| 1+2      | 3                      |
| 5-3      | 2                      |
| 4*6      | 24                     |
| 8/2      | 4                      |
| 123+     | エラー                  |
| 1+*2     | エラー                  |
| 1/0      | エラー                  |
| C        | 0                      |
